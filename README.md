# Infinite Dimensions

Official website for the Infinite Dimensions music project.

## Live Site

Visit the live site here: [https://mysticalg.kesug.com](https://mysticalg.kesug.com)

## Local Files

- `index.html` — main page content
- `styles.css` — site styling
- `script.js` — interactive behavior
- `data/printful-products.json` — static merch catalog data
- `admin.php` — password-protected browser admin to trigger sync/deploy tasks
